## **Line — Visual Timeline Editor**

Line is a C++ application that lets you create, visualize, and manage timelines using a node-based interface. Each node represents an event, and you can connect nodes to show relationships or sequences over time. Every node stores detailed information like event date, title, and description. The software also supports saving and loading timelines, making it easy to manage and revisit your projects.
Which is basically a visual diary kind of software .


### What’s Included 

-  A .zip file which include a Line.exe , .dll file used , and also depended folders.
-  No installation required — just download, unzip and run.
-  Windows support
-  All core Line features available

### New Feature

- Interactive Node-Based Timeline — Add, edit, and connect nodes to build custom timelines.
- Event Details — Each node can store , Details like Date , Event title, Description.
- Save & Load Timelines — Persist your timelines and open them anytime.
- Clean and Intuitive UI — Designed for quick timeline creation and editing.
- Lightweight C++ Application — High performance and minimal dependencies.

### Tech Used
- C++ — Core application logic.
- Qt (QGraphicsView) — For graphics and GUI.
- CMake — Cross-platform build system.
